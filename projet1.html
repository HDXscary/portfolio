ACCUEIL MIEUX ME CONNAÎTRE MON PARCOUR MES PROJETS CV CONTACT

Ce projet est un projet personnel, qui a pour but que, lorsqu'une personne a une requête en informatique, elle puisse aller sur ce site web, remplir le formulaire et enfin me l'envoyer par mail. ce projet a été réalisé avec HTML, CSS et JavaScript. pour le formulaire de contact, j'ai utilisé le protocole web mailto afin de pouvoir envoyer les requêtes par mail. Ce protocole permet de remplir le formulaire et de rediriger chaque valeur de chaque case pour les insérer dans un mail prêt à être envoyé, tout en ouvrant le client mail par défaut de l'utilisateur.
